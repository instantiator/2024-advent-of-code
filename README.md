# Advent of Code 2024

This repository contains my solutions to [Advent of Code 2024](https://adventofcode.com) written in C#. Run a solution with the `run.sh` script:

```bash
./run.sh <day> <part>
```

Data files from each day's puzzle are in: `AdventOfCode2024/Data`
